# 🌸 Music Bouquet v8.4 FIX+

Музыкальный конструктор подарков с букетами, ссылками и изображениями.

---

## 🚀 Возможности

### 💐 Букеты
- 6 встроенных букетов
- выбор кликом
- подсветка активного букета

### 📁 Пользовательские изображения
- загрузка из галереи (file input)
- сохранение в localStorage
- использование как обложки подарка

### ☁️ Dropbox / URL
- поддержка ссылок на изображения
- автоматическое исправление Dropbox (`?dl=0 → ?raw=1`)
